1) install python
2) create and activate virtual environment (python -m venv venv)(venv/Scripts/activate)
3) install all requirements (pip install -r requirements.txt)
4) make migrations (python manage.py makemigrations) (python manage.py migrate)
5) run local server (python manage.py runserver)
6) you can make superuser to access to admin panel (python manage.py createsuperuser) (/admin)
